# PlayerScout

PlayerScout is a single page app built with a Rails API and React/Redux front-end. The purpose of this app is for scouts to keep a roster of possible players.

## Installation

```
cd playerscout-api
rails db:migrate
rails s -p 3001

cd playerscout-client
npm install
npm start

```
